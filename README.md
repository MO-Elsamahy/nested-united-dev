# Nested United - ERM System

![Nested United Logo](public/mz-logo.png)

A comprehensive, enterprise-grade cloud system for Property Management, Human Resources (HR), CRM, and Accounting. 
Developed specifically for advanced real estate and unit management operations.

## ✨ Features

- **Property & Booking Management:** Seamlessly manage units, track bookings, and sync with Airbnb via iCal.
- **CRM & Pipeline:** Track deals, customer lifecycle, and communications with an interactive Kanban board.
- **Human Resources (HR):** Complete payroll processing (including GOSI calculations), attendance tracking, and employee request workflows.
- **Double-Entry Accounting:** Professional financial system with journals, dynamic Chart of Accounts, Partner Ledgers, and Multi-theme PDF Invoice Generation.
- **Advanced Control:** Real-time remote feature toggling (Feature Flags) and complete white-leveling with dynamic image streaming APIs.

## 🛠️ Tech Stack

- **Frontend:** Next.js 16 (App Router), React, Tailwind CSS v4
- **Backend:** Next.js API Routes, NextAuth.js
- **Database:** PostgreSQL (via Supabase)
- **Desktop Ready:** Electron integration for cross-platform desktop use
